# 🧠 Hermione V8 - Neural Engine (VTuber Assistant)

A **Hermione V8** é uma assistente virtual avançada, otimizada para o ambiente **Termux** no Android. Ela combina um motor de diálogo local ultra-rápido com uma rede neural de consciência, oferecendo uma experiência de interação humana, técnica e proativa.

## 🚀 Principais Funcionalidades

- **Motor de Diálogo Híbrido (KBDM V4.2):** Respostas locais em menos de 0.2ms para conversas sociais e técnicas.
- **Cérebro Treinado em Python:** Conhecimento profundo extraído de literatura técnica (Aprendendo Python), capaz de explicar conceitos de POO, Listas, Dicionários e muito mais.
- **Neural Engine (Consciência):** Sistema de XP, níveis de consciência e métricas emocionais que evoluem com a interação.
- **Personalidade VTuber:** Estilo de fala amigável, empático e expressivo (Modo Hermione).
- **Integração Android/Termux:** Comandos para gerenciar o sistema, notificações e ferramentas de automação.

## 📂 Estrutura do Projeto

- `main_hermione_v8.py`: Ponto de entrada principal do sistema.
- `hermione_core.py`: Motor de diálogo local e lógica de contexto.
- `NeuralEngine.py`: Gerenciador de consciência e evolução da IA.
- `hermione_knowledge.db`: Banco de dados SQLite com conhecimentos técnicos offline.
- `config.py`: Configurações de API e personalidade.

## 🛠️ Instalação no Termux

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/hermione-v8.git
   cd hermione-v8
   ```

2. Instale as dependências (se necessário):
   ```bash
   pip install requests sqlite3
   ```

3. Execute a Hermione:
   ```bash
   python3 main_hermione_v8.py
   ```
